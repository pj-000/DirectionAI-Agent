# Exam Generation Skill

## 触发条件

当用户说以下类似的话时，应使用此技能：

- "帮我出题"、"生成试题"
- "做一套 XX 的考试卷"
- "出几道关于 XX 的练习题"
- "设计一个 XX 的测验"
- 用户明确要求创建 exam paper / test / 试题 / 试卷

## 工作流

### Step 1: 理解用户需求

从用户的请求中提取以下信息：

- **topic（主题）**: 考试科目的核心主题（必填）
- **output_language（输出语言）**: 试题语言，默认"中文"
- **difficulty（难度）**: 难度等级
  - 简单：适合基础练习、随堂测验
  - 中等：适合期中期末考试
  - 困难：适合竞赛、选拔考试
- **question_types（题型）**: 需要包含的题型（逗号分隔）
  - 选择题：客观题，适合大题量覆盖知识点
  - 填空题：考察具体知识点掌握
  - 简答题：考察理解和表述能力
  - 论述题：考察综合分析和表达能力
  - 判断题：快速判断对错
- **num_questions（题目数量）**: 总题数，默认 10 题
- **enable_web_search（联网搜索）**: 是否搜索相关资料，默认 False
- **content（附加要求）**: 具体章节、重点知识、特殊要求

### Step 2: 调用 generate_exam 工具

使用提取的参数调用 `generate_exam` 工具：

```python
generate_exam(
    topic="Python 基础",  # 必填
    output_language="中文",
    difficulty="中等",
    question_types="选择题,填空题,简答题",
    num_questions=15,
    enable_web_search=False,
    content="重点考察：变量和数据类型、条件语句、循环结构、函数定义"
)
```

### Step 3: 解释结果

工具返回后，向用户说明：

- 生成的题目数量和题型分布
- 包含的知识点覆盖
- 是否包含参考答案和评分标准

### Step 4: 迭代优化（如需要）

- **调整难度**: 重新指定 difficulty
- **增加特定题型**: 修改 question_types
- **配套 PPT 或教案**: 可结合 `generate_ppt` 或 `generate_lesson_plan` 生成配套教学材料

## 示例对话

```
用户: 我需要一套关于"数据库基础"的试题，包含选择题和简答题，大概 10 道题，难度中等
助手: (调用 generate_exam 工具)
助手: ✅ 试题生成任务已启动！
      主题: 数据库基础
      难度: 中等
      题量: 10 题
      题型: 选择题, 简答题
      📝 生成的试题将包含：客观题（选择）、主观题（简答）、参考答案与评分标准
```

## 注意事项

- **topic 是必填参数**: 如果用户没有明确主题，先询问
- **难度等级**: 影响题目深度和广度，需要与 target_audience 匹配
- **题型组合**: 不同题型考察不同能力，合理组合可以全面评估
- **题目数量**: 建议 10-20 题，太少无法覆盖关键知识点，太多则考试时间过长
- **重点内容**: 在 content 中明确重点章节有助于生成更有针对性的题目

## 与其他技能的配合

- **配合 generate_ppt**: 可以先做配套 PPT 课件，再基于课件内容出题
- **配合 generate_lesson_plan**: 可以先做配套教案，再基于教学目标出题
